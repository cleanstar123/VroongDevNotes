## 1. 출금제한 페이지 로딩 문제 해결

> [!info] 배경
> - 출금제한 관리 페이지 진입 시 "등록된 협력사가 없습니다" 표시
> - 협력사 드롭다운도 비어있음
> - API 호출 (`/api/v1/withdrawal-limit/partner/list`, `/partner/names`)이 무한 대기

> [!warning] 문제
> - SQL 쿼리는 실행되지만 `fetch()` 결과가 반환되지 않음
> - 쿼리 실행 시간이 7초 이상 소요 (정상: 0.5초)
> - `vr_account_mst` 테이블에 `SUPPLY_ID` 인덱스 없음 → Full Table Scan 발생
> - 상관 서브쿼리가 167개 협력사 × 52,769개 레코드마다 실행

**문제의 쿼리 (JOIN 조건):**
```sql
vam.SEQ_NO = (
  SELECT MAX(SEQ_NO) FROM vr_account_mst
  WHERE SUPPLY_ID = vsi.SUPP_ID  -- 협력사마다 실행!
    AND ACCT_TARGET = 'S'
    AND ACCT_TYPE NOT IN ('SD', 'RD')
)
```

> [!success] 해결
> - **인덱스 추가** (운영 DB)
> ```sql
> CREATE INDEX idx_account_mst_supply_target_type
> ON vr_account_mst (SUPPLY_ID, ACCT_TARGET, ACCT_TYPE, SEQ_NO);
> ```
> - 쿼리 성능: 7초 → 0.5초로 개선
> - 페이지 로딩 즉시 완료

> [!tip] 배운 점
> - 상관 서브쿼리가 있는 JOIN은 인덱스가 필수
> - 쿼리가 "갑자기" 느려지거나 빨라지면 Buffer Pool 캐싱 또는 락 경합 의심
> - `SHOW FULL PROCESSLIST`, `information_schema.INNODB_TRX`로 장시간 트랜잭션 확인

### 추가 발견: 장시간 트랜잭션

```
143560248  RUNNING  2026-03-04 06:31:53  1564초 (26분!)
```
- 운영 서버에서 26분간 열려있던 트랜잭션 발견
- 락 경합으로 쿼리 지연 발생 가능성

## 2. relay-server 봇 탐지 쿨다운 로직 추가

> [!info] 배경
> - 세션 만료 후 자동 재인증 시도 시 배민 봇 탐지로 차단
> - 매 분마다 재시도 → 반복 실패로 IP 블랙리스트 강화
> - 14회 연속 실패 로그 발생

> [!warning] 문제
> - 봇 탐지 시에도 1분마다 계속 재시도
> - 반복 시도가 상황을 악화시킴
> - 수동 개입 없이는 복구 불가

```
[15:08:47] [Puppeteer] 봇 탐지 감지 - 세션 삭제 후 재시도 필요
[15:08:47] [Puppeteer] 페이지 내용: 죄송합니다. 올바르지 않은 요청으로 페이지를 보실 수 없습니다.
```

> [!success] 해결
> - `services/auto-reauth.js`에 봇 탐지 쿨다운 로직 추가
> - 쿨다운 시간 (누적에 따라 증가):
>   - 1회: 30분
>   - 2회: 60분
>   - 3회: 90분
>   - 4회+: 120분 (최대)
> - 쿨다운 수동 해제 API 추가

**추가된 기능:**

| 메서드 | 설명 |
|--------|------|
| `_isBotDetectionError()` | 봇 탐지 에러 판별 |
| `_setBotDetectionCooldown()` | 쿨다운 설정 |
| `isInBotDetectionCooldown()` | 쿨다운 중인지 확인 |
| `clearBotDetectionCooldown()` | 쿨다운 수동 해제 |

**추가된 API:**
```
POST /relay/auth/auto-reauth/clear-cooldown
```

**동작 로그:**
```
[AutoReauth] 봇 탐지 쿨다운 설정: 30분 (누적 1회)
[AutoReauth] 쿨다운 해제 시각: 2026. 3. 4. 오후 4:15:00
[AutoReauth] 봇 탐지 쿨다운 중 - 28분 후 재시도 가능
```

### 수정 파일

```
relay-server/services/auto-reauth.js
relay-server/server.js
```
