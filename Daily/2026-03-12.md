## 1. 선정산 배치 시간 변경

> [!info] 배경
> - 선정산 배치(`vrRiderAcctDailyIfJob`)의 실행 시간을 오후에서 오전으로 변경 요청
> - `/api/v1/migration/migrate` API에서 `tableName=vr_rider_acct_daily`로 전날 데이터 처리

### 변경 내역

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 실행 시간 | 14:10 | **11:10** |
| Cron 표현식 | `0 10 14 * * ?` | `0 10 11 * * ?` |
| 트리거 이름 | `vrRiderAcctDailyIfTrigger1410` | `vrRiderAcctDailyIfTrigger1110` |

### 수정 파일

| 파일 | 위치 | 변경 내용 |
|------|------|----------|
| `MigrationQuartzConfig.java` | vroong-3pl-platform-batch | 트리거 시간 변경 |

### 코드 변경

```java
// src/main/java/com/vroong/batch/config/quartz/MigrationQuartzConfig.java

@Bean
public Trigger vrRiderAcctDailyIfTrigger1110() {
  return TriggerBuilder.newTrigger()
      .forJob(vrRiderAcctDailyIfJobDetail())
      .withIdentity("vrRiderAcctDailyIfTrigger1110", "migrationGroup")
      .withDescription("오전 11:10 VR 라이더 계정 일일 정보 마이그레이션")
      .withSchedule(CronScheduleBuilder.cronSchedule("0 10 11 * * ?"))
      .build();
}
```

### DB 변경 쿼리

#### 1. 기존 트리거 삭제

```sql
-- QRTZ_CRON_TRIGGERS에서 삭제
DELETE FROM QRTZ_CRON_TRIGGERS
WHERE SCHED_NAME = 'VroongBatchScheduler'
  AND TRIGGER_NAME = 'vrRiderAcctDailyIfTrigger1410'
  AND TRIGGER_GROUP = 'migrationGroup';

-- QRTZ_TRIGGERS에서 삭제
DELETE FROM QRTZ_TRIGGERS
WHERE SCHED_NAME = 'VroongBatchScheduler'
  AND TRIGGER_NAME = 'vrRiderAcctDailyIfTrigger1410'
  AND TRIGGER_GROUP = 'migrationGroup';
```

#### 2. 새 트리거 수동 추가 (애플리케이션 재시작 없이 적용 시)

```sql
-- QRTZ_JOB_DETAILS에 Job이 없는 경우 먼저 추가
INSERT INTO QRTZ_JOB_DETAILS (
  SCHED_NAME, JOB_NAME, JOB_GROUP, DESCRIPTION, JOB_CLASS_NAME,
  IS_DURABLE, IS_NONCONCURRENT, IS_UPDATE_DATA, REQUESTS_RECOVERY, JOB_DATA
) VALUES (
  'VroongBatchScheduler', 'vrRiderAcctDailyIfJob', 'migrationGroup',
  'VR 라이더 계정 일일 정보 마이그레이션 배치 작업',
  'com.vroong.batch.quartz.BatchQuartzJob',
  1, 0, 0, 0, NULL
);

-- QRTZ_TRIGGERS에 추가
INSERT INTO QRTZ_TRIGGERS (
  SCHED_NAME, TRIGGER_NAME, TRIGGER_GROUP, JOB_NAME, JOB_GROUP,
  DESCRIPTION, NEXT_FIRE_TIME, PREV_FIRE_TIME, PRIORITY, TRIGGER_STATE,
  TRIGGER_TYPE, START_TIME, END_TIME, CALENDAR_NAME, MISFIRE_INSTR, JOB_DATA
) VALUES (
  'VroongBatchScheduler', 'vrRiderAcctDailyIfTrigger1110', 'migrationGroup',
  'vrRiderAcctDailyIfJob', 'migrationGroup',
  '오전 11:10 VR 라이더 계정 일일 정보 마이그레이션',
  NULL, -1, 5, 'WAITING',
  'CRON', UNIX_TIMESTAMP() * 1000, 0, NULL, 0, NULL
);

-- QRTZ_CRON_TRIGGERS에 추가
INSERT INTO QRTZ_CRON_TRIGGERS (
  SCHED_NAME, TRIGGER_NAME, TRIGGER_GROUP, CRON_EXPRESSION, TIME_ZONE_ID
) VALUES (
  'VroongBatchScheduler', 'vrRiderAcctDailyIfTrigger1110', 'migrationGroup',
  '0 10 11 * * ?', 'Asia/Seoul'
);
```

> [!tip] 참고
> - 기존 트리거 삭제 후 애플리케이션 재시작하면 새 트리거가 자동 등록됨
> - 수동 INSERT 시 `QRTZ_JOB_DETAILS` → `QRTZ_TRIGGERS` → `QRTZ_CRON_TRIGGERS` 순서로 외래키 관계 있음
