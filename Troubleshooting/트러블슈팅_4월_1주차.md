## 2. 원인 분석

### 2.1 문제의 본질

- 현재 운영 DB가 **지원 종료 예정 버전(MySQL 8.0)** 위에서 동작 중
- AWS 정책상 일정 이후:
    - 보안 패치 미제공
    - 비용 증가 또는 강제 업그레이드 발생

---

### 2.2 문제 발생 조건

- 운영 DB가 MySQL 8.0 사용 중
- 업그레이드 계획 없이 유지할 경우

---

### 2.3 발생 가능한 문제

#### 1. 비용 증가

- Extended Support 사용 시 추가 요금 발생

#### 2. 자동 업그레이드 리스크

- 유지보수 시간 중 자동 업그레이드
- DB 재시작 → 서비스 장애 가능

#### 3. 애플리케이션 호환성 문제

- JPA/Hibernate 동작 변화
- SQL 문법 충돌
- 쿼리 성능 변화

---
