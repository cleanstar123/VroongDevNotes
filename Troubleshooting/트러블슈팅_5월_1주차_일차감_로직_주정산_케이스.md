# 주정산 라이더 일차감 프로세스 안내

---

## 개요

주정산 라이더는 **매주 1회 정산** 시 배달료를 지급받습니다.  
이때 계약에 따라 설정된 **일차감이 주간 배달료에서 차감**된 후 지급됩니다.

일차감은 계약 규칙(`vr_deduct_mst`)을 기반으로 **이론값으로 계산**됩니다.

> 아래 모든 예시 기준: 일차감 단가 **10,000원 / 근무일** (근무일 기준 타입)

---

## 일차감 계산 방식

| 유형 | 설명 | 예시 |
|---|---|---|
| **근무일 기준** (WORK_DAY) | 해당 주 실제 배달을 수행한 날 수 × 단가 | 5일 × 10,000원 = 50,000원 |
| **일수 기준** (DAILY) | 정산 기간의 달력 일수 × 단가 | 7일 × 1,500원 = 10,500원 |

---

## 기본 구조

```
주간 총 배달료
- 일차감 (먼저 적용)
= 실지급액
- 이월 미수금 회수 (실지급액에서 가능한 만큼)
= 실제 이체액
```

> 선정산과 처리 순서가 다릅니다.  
> 선정산: 미수금 회수 → 일차감  
> **주정산: 일차감 → 미수금 회수**

---

## 케이스별 예시

---

### Case 1. 정상 차감 (미수금 없음)

> 주간 배달료가 일차감보다 충분하고 이월 미수금도 없는 경우

| 항목 | 금액 |
|---|---|
| 주간 배달료 | 500,000원 |
| 일차감 (5일 × 10,000원) | 50,000원 |
| 실지급액 | 450,000원 |
| 이월 미수금 | 없음 |
| **실제 이체액** | **450,000원** |
| 미수금 발생 | 없음 |

→ 일차감 전액이 차감되고 450,000원이 지급됩니다.

---

### Case 2. 부분 차감 + 유보

> 주간 배달료가 일차감보다 적지만 0보다는 큰 경우

| 항목 | 금액 |
|---|---|
| 주간 배달료 | 30,000원 |
| 일차감 | 50,000원 |
| 실제 적용된 차감 | 30,000원 (가능한 금액만) |
| 실지급액 | 0원 |
| **실제 이체액** | **0원** |
| **미수금 (다음 주로 이월)** | **20,000원** |

→ 배달료 전액(30,000원)까지만 차감하고, 부족한 **20,000원은 미수금으로 다음 주 이월**됩니다.  
→ 실지급액은 0원 미만으로 내려가지 않습니다.

---

### Case 3. 전액 유보 (배달 없는 주)

> 해당 주에 배달이 없어 배달료가 0인 경우

| 항목 | 금액 |
|---|---|
| 주간 배달료 | 0원 |
| 일차감 | 50,000원 |
| 실제 적용된 차감 | 0원 |
| 실지급액 | 0원 |
| **실제 이체액** | **0원** |
| **미수금 (다음 주로 이월)** | **50,000원** |

→ 차감할 배달료가 없으므로 일차감 **전액이 미수금으로 다음 주 이월**됩니다.

---

### Case 4. 이월 미수금 — 실지급액에서 전액 회수

> 이전 주 미수금이 있고, 이번 주 실지급액이 충분한 경우

| 항목 | 금액 |
|---|---|
| 이월 미수금 | 20,000원 |
| 주간 배달료 | 500,000원 |
| 일차감 | 50,000원 |
| 실지급액 (일차감 후) | 450,000원 |
| 이월 미수금 회수 | 20,000원 (실지급액에서 차감) |
| **실제 이체액** | **430,000원** |
| 미수금 잔여 | 0원 (완전 회수) |

→ 일차감을 먼저 적용한 뒤, 실지급액(450,000원)에서 이월 미수금(20,000원)을 회수합니다.

---

### Case 5. 이월 미수금 — 실지급액이 부족하여 일부만 회수 + 재이월

> 미수금이 크고 실지급액이 이를 모두 커버하지 못하는 경우

| 항목 | 금액 |
|---|---|
| 이월 미수금 | 500,000원 |
| 주간 배달료 | 300,000원 |
| 일차감 | 50,000원 |
| 실지급액 (일차감 후) | 250,000원 |
| 이월 미수금 회수 | 250,000원 (실지급액 전액) |
| **실제 이체액** | **0원** |
| **미수금 재이월** | **250,000원** (미회수 잔여) |

→ 실지급액 전액(250,000원)이 미수금 회수에 사용되고, 라이더에게는 이체되는 금액이 없습니다.  
→ 회수하지 못한 250,000원은 다음 주로 재이월됩니다.

---

### Case 6. 이월 미수금 + 이번 주도 유보 발생 → 미수금 누적

> 이월 미수금이 있는데 이번 주 배달료도 부족하여 새 유보까지 발생하는 경우

| 항목 | 금액 |
|---|---|
| 이월 미수금 | 30,000원 |
| 주간 배달료 | 20,000원 |
| 일차감 | 50,000원 |
| 실제 적용된 차감 | 20,000원 (배달료 전액) |
| 새 유보 | 30,000원 |
| 실지급액 | 0원 |
| 이월 미수금 회수 | 0원 (실지급액 없으므로 회수 불가) |
| **실제 이체액** | **0원** |
| **미수금 재이월** | **60,000원** (이전 30,000원 + 새 유보 30,000원) |

→ 실지급액이 0원이므로 이전 미수금을 회수할 수 없고, 이번 주 새 유보까지 더해져 미수금이 **누적**됩니다.

---

## 전체 흐름 요약

```
주정산 실행 (주 1회)
    │
    ├─ 일차감 계산 (vr_deduct_mst 규칙 기반 이론값)
    │       근무일 기준: 실제 근무일 수 × 단가
    │       일수 기준:   정산 기간 일수 × 단가
    │
    ├─ 주간 배달료 ≥ 일차감?
    │       YES → 전액 차감                       (Case 1)
    │       NO  → 가능한 금액만 차감, 나머지 유보  (Case 2, 3)
    │
    ├─ 이월 미수금 있음?
    │       YES → 실지급액에서 가능한 만큼 회수
    │               실지급액 ≥ 미수금 → 전액 회수  (Case 4)
    │               실지급액 < 미수금 → 일부 회수, 나머지 재이월  (Case 5, 6)
    │
    └─ 실제 이체 (항상 0원 이상)
```

---

## 선정산과의 주요 차이점

| 항목 | 주정산 | 선정산 |
|---|---|---|
| 정산 주기 | 주 1회 | 매일 |
| 일차감 계산 | 규칙 기반 이론값 | 실제 적용된 실적값 |
| 처리 순서 | **일차감 먼저 → 미수금 회수** | **미수금 회수 먼저 → 일차감** |
| 잔액 기반 미수금 회수 | 없음 | 있음 (Full-or-Nothing) |

---

## 핵심 원칙

1. **실지급액은 절대 음수가 되지 않습니다.**
2. **일차감이 미수금 회수보다 먼저 적용됩니다.**
3. **미수금은 실지급액 범위 내에서만 회수됩니다.** (잔액 차감 없음)
4. **미수금은 해소될 때까지 매주 이월됩니다.**
