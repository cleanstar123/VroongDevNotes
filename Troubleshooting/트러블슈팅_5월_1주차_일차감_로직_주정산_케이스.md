# 주정산 일차감 프로세스 안내

> 주정산은 라이더 유형에 따라 계산 방식이 다릅니다.  
> - **선정산 라이더의 주정산**: 선정산에서 이미 일차감이 처리됨, 주정산은 정산 잔액 정리
> - **주정산 라이더의 주정산**: 주정산에서 일차감 전체를 한 번에 처리

---

# Part 1. 선정산 라이더의 주정산

## 개요

선정산 라이더는 **매일 배달료를 미리 지급**받으며, 이 과정에서 일차감이 매일 적용됩니다.  
주정산은 **일주일 치 배달료와 선정산 지급액을 최종 정리**하는 과정으로, 대부분 실지급액은 0원입니다.

주정산 일차감 금액은 **선정산에서 실제 적용된 금액 합계(실적값)** 를 사용합니다.

> 아래 모든 예시 기준: 일차감 단가 **7,000원 / 근무일**

---

## 기본 구조

```
주간 총 배달료
- 선정산 지급액 (매일 지급된 금액 합계)
= 주정산 정산 가능 금액
- 선정산 실적 일차감 (선정산에서 실제 차감된 금액 합계)
= 주정산 실지급액 (대부분 0원)
- 이월 미수금 회수 (있는 경우)
= 실제 이체액
```

---

## 케이스별 예시

---

### Case A-1. 선정산에서 모든 일차감 정상 처리 → 주정산 실지급 0원

> 5일 모두 배달료가 충분하여 선정산에서 전액 차감된 경우

**선정산 내역 (일별)**

| 요일 | 배달료 | 선정산 지급 | 일차감 적용 | 유보 |
|---|---|---|---|---|
| 월 | 80,000원 | 73,000원 | 7,000원 | - |
| 화 | 80,000원 | 73,000원 | 7,000원 | - |
| 수 | 80,000원 | 73,000원 | 7,000원 | - |
| 목 | 80,000원 | 73,000원 | 7,000원 | - |
| 금 | 80,000원 | 73,000원 | 7,000원 | - |
| **합계** | **400,000원** | **365,000원** | **35,000원** | **0원** |

**주정산**

| 항목 | 금액 |
|---|---|
| 주간 총 배달료 | 400,000원 |
| 선정산 지급액 | 365,000원 |
| 주정산 정산 가능 금액 | 35,000원 |
| 선정산 실적 일차감 | 35,000원 |
| 주정산 실지급액 | **0원** |
| 이월 미수금 | 없음 |
| **실제 이체액** | **0원** |

→ 선정산에서 모든 일차감이 처리되었으므로, 주정산 정산 가능 금액(35,000원)이 실적 일차감(35,000원)과 일치하여 실지급 0원이 됩니다.

### Case A-2. 특정일 선정산 유보 발생 → 주정산 이월 미수금 처리

> 금요일 배달료가 부족하여 선정산에서 일부 유보된 경우

**선정산 내역 (일별)**

| 요일     | 배달료          | 선정산 지급       | 일차감 적용      | 유보         |

| 수      | 80,000원      | 73,000원      | 7,000원      | -          |
| 목      | 80,000원      | 73,000원      | 7,000원      | -          |
| 금      | 2,000원       | 0원           | 2,000원      | **5,000원** |
| **합계** | **322,000원** | **292,000원** | **30,000원** | **5,000원** |

**주정산**

| 항목              | 금액       |

→ 금요일 유보 **5,000원은 이월 미수금**으로 남아 선정산 잔액 기반 회수 또는 다음 주 주정산에서 처리됩니다.

### Case A-3. 선정산 유보 + 주정산에서도 차감 일부 미처리 → 미수금 누적

> 배달 없는 날이 있어 선정산 유보가 발생하고, 주정산에서도 처리되지 못한 경우

| 수      | 80,000원      | 73,000원      | 7,000원      | -          |
| 목      | 80,000원      | 73,000원      | 7,000원      | -          |
| 금      | 0원           | 0원           | 0원          | **7,000원** |
| **합계** | **320,000원** | **292,000원** | **28,000원** | **7,000원** |

**주정산**

| 항목              | 금액       |

→ 금요일 유보 **7,000원은 이월 미수금**으로, 선정산 잔액 기반 회수(Full-or-Nothing) 또는 다음 주로 재이월됩니다.

---

### Case A-4. 이월 미수금 + 이번 주도 정상 정산

> 지난 주 유보 미수금이 있고, 이번 주 주정산 실지급액으로 회수하는 경우

| 항목           | 금액                    |
| ------------ | --------------------- |
| 이월 미수금       | 7,000원                |
| 주간 총 배달료     | 400,000원              |
| 선정산 지급액      | 365,000원              |
| 주정산 정산 가능 금액 | 35,000원               |
| 선정산 실적 일차감   | 35,000원               |
| 주정산 실지급액     | 0원                    |
| 이월 미수금 회수    | 0원 (실지급액이 0원이라 회수 불가) |
| **실제 이체액**   | **0원**                |
| **미수금 재이월**  | **7,000원**            |

→ 주정산 실지급액이 0원이면 이월 미수금을 회수할 수 없어 **다음 주로 재이월**됩니다.  
→ 선정산에서 잔액 기반 회수(Full-or-Nothing)가 성공해야 미수금이 해소됩니다.

# Part 2. 주정산 라이더의 주정산

## 개요

주정산 라이더는 **매주 1회 정산** 시에만 배달료를 지급받습니다.  
일차감은 계약 규칙(`vr_deduct_mst`)을 기반으로 **이론값으로 계산**되어 주정산 배달료에서 차감됩니다.

> 아래 모든 예시 기준: 일차감 단가 **10,000원 / 근무일** (근무일 기준 타입)

---

## 기본 구조

```
주간 총 배달료
- 일차감 (먼저 적용, 이론값)
= 실지급액
- 이월 미수금 회수 (실지급액에서 가능한 만큼)
= 실제 이체액
```

---

## 케이스별 예시

---

### Case B-1. 정상 차감 (미수금 없음)

> 주간 배달료가 일차감보다 충분하고 이월 미수금도 없는 경우

| 항목                 | 금액           |
| ------------------ | ------------ |
| 주간 배달료             | 500,000원     |
| 일차감 (5일 × 10,000원) | 50,000원      |
| 실지급액               | 450,000원     |
| 이월 미수금             | 없음           |
| **실제 이체액**         | **450,000원** |
| 미수금 발생             | 없음           |

→ 일차감 전액이 차감되고 450,000원이 지급됩니다.

---

### Case B-2. 부분 차감 + 유보

> 주간 배달료가 일차감보다 적지만 0보다는 큰 경우

| 항목                 | 금액                |
| ------------------ | ----------------- |
| 주간 배달료             | 30,000원           |
| 일차감                | 50,000원           |
| 실제 적용된 차감          | 30,000원 (가능한 금액만) |
| 실지급액               | 0원                |
| **실제 이체액**         | **0원**            |
| **미수금 (다음 주로 이월)** | **20,000원**       |

→ 배달료 전액(30,000원)까지만 차감하고, 부족한 **20,000원은 미수금으로 다음 주 이월**됩니다.

---

### Case B-3. 전액 유보 (배달 없는 주)

> 해당 주에 배달이 없어 배달료가 0원인 경우

| 항목                 | 금액          |
| ------------------ | ----------- |
| 주간 배달료             | 0원          |
| 일차감                | 50,000원     |
| 실제 적용된 차감          | 0원          |
| 실지급액               | 0원          |
| **실제 이체액**         | **0원**      |
| **미수금 (다음 주로 이월)** | **50,000원** |

→ 차감할 배달료가 없으므로 일차감 **전액이 미수금으로 다음 주 이월**됩니다.

---

### Case B-4. 이월 미수금 — 실지급액에서 전액 회수

> 이전 주 미수금이 있고, 이번 주 실지급액이 충분한 경우

| 항목           | 금액                  |
| ------------ | ------------------- |
| 이월 미수금       | 20,000원             |
| 주간 배달료       | 500,000원            |
| 일차감          | 50,000원             |
| 실지급액 (일차감 후) | 450,000원            |
| 이월 미수금 회수    | 20,000원 (실지급액에서 차감) |
| **실제 이체액**   | **430,000원**        |
| 미수금 잔여       | 0원 (완전 회수)          |

→ 일차감 먼저 적용 후, 실지급액(450,000원)에서 이월 미수금(20,000원)을 회수합니다.

---

### Case B-5. 이월 미수금 — 실지급액이 부족하여 일부만 회수 + 재이월

> 미수금이 커서 실지급액으로 전액 회수가 불가한 경우

| 항목           | 금액                    |
| ------------ | --------------------- |
| 이월 미수금       | 500,000원              |
| 주간 배달료       | 300,000원              |
| 일차감          | 50,000원               |
| 실지급액 (일차감 후) | 250,000원              |
| 이월 미수금 회수    | 250,000원 (실지급액 전액)    |
| **실제 이체액**   | **0원**                |
| **미수금 재이월**  | **250,000원** (미회수 잔여) |

→ 실지급액 전액이 미수금 회수에 사용되고, 미회수 잔여는 다음 주로 재이월됩니다.

---

### Case B-6. 이월 미수금 + 이번 주도 유보 발생 → 미수금 누적

> 이월 미수금이 있는데 이번 주 배달료도 부족하여 새 유보까지 발생하는 경우

| 항목          | 금액                                      |
| ----------- | --------------------------------------- |
| 이월 미수금      | 30,000원                                 |
| 주간 배달료      | 20,000원                                 |
| 일차감         | 50,000원                                 |
| 실제 적용된 차감   | 20,000원 (배달료 전액)                        |
| 새 유보        | 30,000원                                 |
| 실지급액        | 0원                                      |
| 이월 미수금 회수   | 0원 (실지급액 없음)                            |
| **실제 이체액**  | **0원**                                  |
| **미수금 재이월** | **60,000원** (이전 30,000원 + 새 유보 30,000원) |

→ 실지급액이 없어 이전 미수금도 회수 불가, 이번 주 유보까지 더해져 미수금이 **누적**됩니다.

# 두 유형 비교 요약

| 항목 | 선정산 라이더의 주정산 | 주정산 라이더의 주정산 |
|---|---|---|
| 일차감 계산 기준 | **실적값** (선정산에서 실제 차감된 금액) | **이론값** (계약 규칙 기반 계산) |
| 정산 가능 금액 | 주간 배달료 - 선정산 지급액 (소액) | 주간 배달료 전액 |
| 주정산 실지급액 | 대부분 **0원** | 배달료 - 일차감 |
| 이월 미수금 회수 | 실지급액이 0원이면 회수 불가 → 선정산 잔액 기반 회수로 처리 | 실지급액에서 가능한 만큼 회수 |
| 잔액 기반 회수 | 선정산(매일) 단계에서 처리 | 없음 |

---

## 핵심 원칙

1. **실지급액은 절대 음수가 되지 않습니다.**
2. **일차감이 이월 미수금 회수보다 먼저 적용됩니다.**
3. **미수금은 실지급액 범위 내에서만 회수됩니다.** (주정산에서는 잔액 차감 없음)
4. **선정산 라이더의 유보 미수금은 선정산 잔액 기반 회수(Full-or-Nothing)로 우선 처리됩니다.**
5. **미수금은 해소될 때까지 매주 이월됩니다.**
