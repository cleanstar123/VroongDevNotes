# 선정산 라이더 일차감 프로세스 안내

---

## 개요

선정산 라이더는 **매일 배달 완료 시점에 배달료를 지급**받습니다.  
이때 계약에 따라 설정된 **일차감(보험료, 장비 대여료 등)이 매일 배달료에서 차감**된 후 지급됩니다.

> 아래 모든 예시 기준: 일차감 단가 **7,000원 / 근무일**

---

## 기본 구조

```
① 이월 미수금 회수 (있는 경우, 오늘 배달료에서 가능한 만큼 우선 회수)
② 회수 후 남은 배달료에서 오늘 일차감 부분차감 (가능한 만큼 차감, 나머지는 미수금)
= 실지급액

③ 부분차감 후 남은 미수금(오늘 미차감분)이 발생한 경우:
   Vroong 계좌 잔액 ≥ 오늘 미차감분 전액 → 잔액에서 전액 차감 (부분 차감 없음)
   Vroong 계좌 잔액 < 오늘 미차감분 전액 → 다음날로 이월 (미수금 누적)

④ ③ 처리 후에도 계좌 잔액이 남아 있고, ①에서 배달료로 미회수된 이월 미수금이 있는 경우:
   남은 잔액 범위 내에서 미회수 이월 미수금을 추가 회수 (부분 회수 허용)
```

> **회수 우선**: 이월 미수금은 오늘 배달료에서 **먼저** 회수됩니다.  
> **부분차감**: 잔여 배달료가 일차감보다 부족하면 가능한 만큼만 차감하고, 나머지는 미수금으로 이월합니다.  
> **잔액 처리 ③**: 오늘 발생한 신규 미수금에 대해 잔액이 전액 이상이면 잔액에서 전액 차감합니다.  
> **잔액 처리 ④**: ③ 이후 남은 잔액은 이전 이월 미수금의 미회수 잔여분을 부분 회수하는 데 사용됩니다.

---

## 케이스별 예시

---

### Case 1. 정상 차감

> 오늘 배달료가 일차감보다 충분한 경우

| 항목       | 금액          |
| -------- | ----------- |
| 이월 미수금   | 0원          |
| 오늘 배달료   | 80,000원     |
| 일차감      | 7,000원      |
| **실지급액** | **73,000원** |
| 신규 미수금   | 없음          |

→ 일차감 전액이 차감되고 73,000원이 지급됩니다.

---

### Case 2. 배달료 부족 — 일차감 부분차감

> 오늘 배달료가 일차감보다 적어 가능한 만큼만 차감되는 경우

| 항목         | 금액                |
| ---------- | ----------------- |
| 이월 미수금     | 0원                |
| 오늘 배달료     | 5,000원            |
| 일차감        | 7,000원            |
| 실제 적용된 차감  | **5,000원** (부분차감) |
| **실지급액**   | **0원**            |
| **신규 미수금** | **2,000원**        |

→ 배달료(5,000원) 전액을 일차감으로 차감하고 실지급액은 0원입니다.  
→ 차감하지 못한 2,000원이 미수금으로 다음날로 이월됩니다.

---

### Case 3. 배달 없는 날 — 전액 미수금

> 오늘 배달이 없거나 배달료가 0인 경우

| 항목         | 금액         |
| ---------- | ---------- |
| 이월 미수금     | 0원         |
| 오늘 배달료     | 0원         |
| 일차감        | 7,000원     |
| 실제 적용된 차감  | 0원         |
| **실지급액**   | **0원**     |
| **신규 미수금** | **7,000원** |

→ 차감할 배달료가 없으므로 일차감 **전액이 미수금으로 이월**됩니다.  
→ 계좌 잔액이 충분하면 잔액에서 별도 차감, 부족하면 미수금으로 다음날 이월됩니다.

---

### Case 4. 이월 미수금 — 배달료에서 전액 회수 후 일차감 정상 차감

> 전날 미수금이 있고, 오늘 배달료가 충분한 경우

| 항목       | 금액                   |
| -------- | -------------------- |
| 이월 미수금   | 2,000원               |
| 오늘 배달료   | 80,000원              |
| 미수금 회수   | 2,000원 (배달료에서 먼저 차감) |
| 회수 후 잔여  | 78,000원              |
| 일차감      | 7,000원 → 전액 차감       |
| **실지급액** | **71,000원**          |
| 미수금 잔여   | 0원 (완전 회수)           |

→ 이월 미수금(2,000원)을 배달료에서 먼저 회수한 후, 잔여 배달료(78,000원)에서 일차감(7,000원)을 차감합니다.

---

### Case 5. 이월 미수금 — 미수금 일부 회수 후 일차감 부분차감

> 이월 미수금이 있는데 오늘 배달료도 부족한 경우 (계좌 잔액 없음 가정)

| 항목            | 금액                                         |
| ------------- | ------------------------------------------ |
| 이월 미수금        | 7,000원                                     |
| 오늘 배달료        | 3,000원                                     |
| 미수금 회수 ①      | 3,000원 (배달료 전액으로 회수)                      |
| 회수 후 잔여       | 0원                                         |
| 일차감 ②         | 7,000원 → 잔여 배달료 0원으로 차감 불가 (전액 미수금)       |
| **실지급액**      | **0원**                                     |
| Vroong 계좌 잔액  | 0원                                         |
| 잔액 차감 ③       | 0원 (잔액 < 신규 미수금 7,000원, 전액 이월)             |
| 잔액 추가회수 ④     | 0원 (잔액 없음)                                 |
| **미수금 재이월**   | **11,000원** (미회수 4,000원 + 오늘 일차감 7,000원) |

→ 미수금 7,000원 중 3,000원만 회수되고 4,000원이 재이월됩니다.  
→ 잔여 배달료가 0원이므로 오늘 일차감 7,000원은 전액 미수금으로 이월됩니다.  
→ 계좌 잔액이 있는 경우는 아래 Case 8을 참조하세요.

---

### Case 6. 오늘 미수금 — 계좌 잔액으로 전액 처리

> 오늘 일차감이 부분 또는 전액 미차감되었지만 Vroong 계좌 잔액이 충분한 경우  
> **조건: 계좌 잔액 ≥ 오늘 신규 미수금 전액이어야만 차감 (부분 차감 없음)**

| 항목           | 금액                            |
| ------------ | ----------------------------- |
| 오늘 배달료       | 0원 (배달 없음)                    |
| 이월 미수금       | 0원                            |
| 일차감          | 7,000원 → 전액 미수금 (신규 7,000원)   |
| Vroong 계좌 잔액 | **10,000원** (≥ 신규 미수금 7,000원) |
| 잔액에서 차감      | **7,000원**                    |
| **실지급액**     | **0원**                        |
| 미수금 발생       | **없음** (잔액으로 완전 처리)           |

→ 오늘 배달료로 일차감을 처리하지 못했더라도, **잔액이 오늘 미수금 전액 이상이면 잔액에서 전액 차감**하여 미수금 이월을 방지합니다.  
→ 라이더의 지급액(TOT_ACCT_VAL)은 변경되지 않고, 차감은 잔액에서만 처리됩니다.

---

### Case 7. 오늘 미수금 — 계좌 잔액도 부족하여 이월

> 오늘 일차감이 미차감되었고 Vroong 계좌 잔액도 부족한 경우

| 항목           | 금액                           |
| ------------ | ---------------------------- |
| 오늘 배달료       | 0원                           |
| 일차감          | 7,000원 → 전액 미수금 (신규 7,000원)  |
| Vroong 계좌 잔액 | **3,000원** (< 신규 미수금 7,000원) |
| 잔액에서 차감      | **0원** (부분 차감 없음)            |
| **미수금 이월**   | **7,000원**                   |

→ 잔액이 오늘 미수금에 미치지 못하면 **이번에는 잔액 차감도 하지 않고 전액을 다음날로 이월**합니다.  
→ 3,000원을 부분 차감하는 방식은 사용하지 않습니다. (계산 일관성 유지)

---

### Case 8. 이월 미수금 — 잔액으로 오늘 미수금 처리 불가하지만 이월 미수금 잔여분 부분 회수

> Case 5 상황에서 계좌 잔액이 일부 있어, ③ 처리 후 ④ 잔여 잔액으로 이월 미수금 잔여분을 부분 회수하는 경우

| 항목            | 금액                                         |
| ------------- | ------------------------------------------ |
| 이월 미수금        | 7,000원                                     |
| 오늘 배달료        | 3,000원                                     |
| 미수금 회수 ①      | 3,000원 (배달료 전액으로 회수)                      |
| 회수 후 잔여       | 0원                                         |
| 일차감 ②         | 7,000원 → 잔여 배달료 0원으로 차감 불가 (전액 미수금)       |
| **실지급액**      | **0원**                                     |
| Vroong 계좌 잔액  | **5,000원**                                 |
| 잔액 차감 ③       | 0원 (잔액 5,000원 < 신규 미수금 7,000원 → full-or-nothing) |
| 잔액 추가회수 ④     | **4,000원** (잔액 5,000원 중 미회수 이월 미수금 4,000원 부분 회수) |
| **미수금 재이월**   | **7,000원** (오늘 일차감 미차감분만 이월)              |

→ 잔액(5,000원)이 오늘 신규 미수금(7,000원)보다 부족하여 ③에서 잔액 차감을 하지 않습니다.  
→ 그러나 ③ 이후 남은 잔액(5,000원)으로 배달료로 미회수된 이월 미수금 잔여분(4,000원)을 **부분 회수**합니다.  
→ 결과적으로 이전 이월 미수금 7,000원은 완전히 회수(3,000원 + 4,000원)되고, 오늘 일차감 7,000원만 남습니다.

---

## 전체 흐름 요약

```
매일 선정산 실행
    │
    ├─ ① 이월 미수금 있음? (오늘 배달료에서 먼저 회수)
    │       YES, 배달료 ≥ 미수금 → 전액 회수              (Case 4)
    │       YES, 배달료 < 미수금 → 가능한 만큼만 회수, 나머지 잔여  (Case 5, 8)
    │       NO  → 생략
    │
    ├─ ② 오늘 일차감 부분차감 (이월 미수금 회수 후 잔여 배달료 기준)
    │       잔여 배달료 ≥ 일차감 → 전액 차감               (Case 1)
    │       잔여 배달료 < 일차감 → 가능한 만큼 차감, 나머지 미수금  (Case 2, 3, 5, 8)
    │
    └─ 실지급 (항상 0원 이상)

    ③ 오늘 신규 미수금(오늘 미차감분)이 있는 경우:
        계좌 잔액 ≥ 미수금 전액 → 잔액에서 전액 차감      (Case 6)
        계좌 잔액 < 미수금 전액 → 재이월 (다음날 미수금 누적) (Case 7, 8)

    ④ ①에서 배달료로 미회수된 이월 미수금 잔여분이 있고, ③ 처리 후 계좌 잔액이 남은 경우:
        남은 잔액 범위 내에서 이월 미수금 잔여분 부분 회수 (부분 회수 허용) (Case 8)
```

---

## 핵심 원칙

1. **실지급액은 절대 음수가 되지 않습니다.**
2. **이월 미수금 회수는 오늘 일차감보다 먼저 처리됩니다.** (배달료에서 우선 회수)
3. **일차감은 부분차감 방식**: 잔여 배달료 범위 내에서 가능한 만큼 차감하고, 나머지는 미수금으로 이월합니다.
4. **잔액 기반 차감은 오늘 신규 미수금에 대해서만 적용되며, 잔액이 미수금 전액 이상일 때만 실행됩니다.** (부분 차감 없음)
5. **미수금은 해소될 때까지 매일 이월됩니다.**
