# 선정산 라이더 일차감 프로세스 안내

---

## 개요

선정산 라이더는 **매일 배달 완료 시점에 배달료를 지급**받습니다.  
이때 계약에 따라 설정된 **일차감(보험료, 장비 대여료 등)이 매일 배달료에서 차감**된 후 지급됩니다.

> 아래 모든 예시 기준: 일차감 단가 **7,000원 / 근무일**

---

## 기본 구조

```
오늘 배달료
- 오늘 일차감 (각 규칙별 전부-or-전무, Greedy 방식)
= 일차감 후 잔여 배달료
- 이월 미수금 회수 (있는 경우, 잔여 배달료에서 가능한 만큼)
= 실지급액

※ 오늘 미처 차감하지 못한 일차감(신규 유보분)이 발생한 경우:
   Vroong 계좌 잔액 ≥ 신규 유보분 전액 → 잔액에서 전액 차감 (부분 차감 없음)
   Vroong 계좌 잔액 < 신규 유보분 전액 → 다음날로 재이월 (미수금 누적)
```

> **핵심**: 일차감은 배달료에서 **먼저** 처리되고, 이월 미수금은 그 이후 잔여 배달료에서 회수됩니다.  
> 일차감 규칙은 **규칙 단위로 전액 차감하거나 전액 유보**합니다. 한 규칙을 부분 적용하지 않습니다.

---

## 케이스별 예시

---

### Case 1. 정상 차감

> 오늘 배달료가 일차감보다 충분한 경우

| 항목       | 금액          |
| -------- | ----------- |
| 오늘 배달료   | 80,000원     |
| 일차감      | 7,000원      |
| **실지급액** | **73,000원** |
| 신규 유보분   | 없음          |

→ 일차감 전액이 차감되고 73,000원이 지급됩니다.

---

### Case 2. 배달료 부족 — 일차감 전액 유보 (Full-or-Nothing)

> 오늘 배달료가 일차감보다 적어 규칙을 적용할 수 없는 경우  
> **규칙은 전부-or-전무: 배달료가 부족하면 해당 규칙 전액이 유보됩니다**

| 항목         | 금액             |
| ---------- | -------------- |
| 오늘 배달료     | 5,000원         |
| 일차감        | 7,000원         |
| 실제 적용된 차감  | **0원** (전액 유보) |
| **실지급액**   | **5,000원**     |
| **신규 유보분** | **7,000원**     |

→ 배달료(5,000원)가 일차감(7,000원)보다 적으므로 해당 규칙은 **전액 유보**됩니다.  
→ 라이더는 오늘 배달료 5,000원을 그대로 지급받습니다.  
→ 유보된 7,000원은 계좌 잔액이 충분하면 잔액에서 별도 차감, 부족하면 미수금으로 다음날 이월됩니다.

---

### Case 3. 전액 유보 (배달 없는 날)

> 오늘 배달이 없거나 배달료가 0인 경우

| 항목             | 금액         |
| -------------- | ---------- |
| 오늘 배달료         | 0원         |
| 일차감            | 7,000원     |
| 실제 적용된 차감      | 0원         |
| **실지급액**       | **0원**     |
| **신규 유보분**     | **7,000원** |

→ 차감할 배달료가 없으므로 일차감 **전액이 유보**됩니다.  
→ 계좌 잔액이 충분하면 잔액에서 별도 차감, 부족하면 미수금으로 다음날 이월됩니다.

---

### Case 4. 이월 미수금 — 배달료에서 전액 회수

> 전날 미수금이 있고, 오늘 배달료가 충분한 경우

| 항목        | 금액                   |
| --------- | -------------------- |
| 이월 미수금    | 2,000원               |
| 오늘 배달료    | 80,000원              |
| 일차감       | 7,000원 → 전액 차감       |
| 일차감 후 잔여  | 73,000원              |
| 이월 미수금 회수 | 2,000원 (잔여 배달료에서 차감) |
| **실지급액**  | **71,000원**          |
| 미수금 잔여    | 0원 (완전 회수)           |

→ 일차감(7,000원)을 먼저 차감한 후, 잔여 배달료(73,000원)에서 이월 미수금(2,000원)을 회수합니다.

---

### Case 5. 이월 미수금 — 배달료가 부족하여 일부만 회수 후 재이월

> 이월 미수금이 있는데 오늘 배달료도 부족한 경우 (잔액도 부족한 시나리오)

| 항목          | 금액                                             |
| ----------- | ---------------------------------------------- |
| 이월 미수금      | 7,000원                                         |
| 오늘 배달료      | 3,000원                                         |
| 일차감         | 7,000원 → 배달료 부족으로 **전액 유보 (신규 유보분 7,000원)**    |
| 일차감 후 잔여    | 3,000원 (차감 없으므로 배달료 그대로)                       |
| 이월 미수금 회수   | 3,000원 (잔여 배달료 전액으로 미수금 회수)                    |
| **실지급액**    | **0원**                                         |
| **미수금 재이월** | **11,000원** (미회수 4,000원 + 신규 유보분 7,000원 이월 합산) |

→ 일차감 규칙(7,000원)은 잔여 배달료(3,000원)로 커버 불가하여 **전액 유보**됩니다.  
→ 이월 미수금 7,000원 중 3,000원만 회수되고 4,000원이 재이월됩니다.  
→ 신규 유보분 7,000원도 계좌 잔액 부족으로 미수금에 합산되어 총 11,000원이 재이월됩니다.

---

### Case 6. 신규 유보분 — 계좌 잔액으로 전액 처리 (Full-or-Nothing)

> 오늘 일차감이 유보되었지만 Vroong 계좌 잔액이 충분한 경우  
> **조건: 계좌 잔액 ≥ 오늘 신규 유보분 전액이어야만 차감 (부분 차감 없음)**

| 항목           | 금액                             |
| ------------ | ------------------------------ |
| 오늘 배달료       | 0원 (배달 없음)                     |
| 일차감          | 7,000원 → 전액 유보 (신규 유보분 7,000원) |
| Vroong 계좌 잔액 | **10,000원** (≥ 신규 유보분 7,000원)  |
| 잔액에서 차감      | **7,000원**                     |
| **실지급액**     | **0원**                         |
| 미수금 발생       | **없음** (잔액으로 완전 처리)            |

→ 오늘 배달료로 일차감을 처리하지 못했더라도, **잔액이 신규 유보분 전액 이상이면 잔액에서 전액 차감**하여 미수금 발생을 방지합니다.  
→ 라이더의 지급액(TOT_ACCT_VAL)은 변경되지 않고, 차감은 잔액에서만 처리됩니다.

---

### Case 7. 신규 유보분 — 계좌 잔액도 부족하여 미수금 이월

> 오늘 일차감이 유보되었고 Vroong 계좌 잔액도 부족한 경우

| 항목           | 금액                             |
| ------------ | ------------------------------ |
| 오늘 배달료       | 0원                             |
| 일차감          | 7,000원 → 전액 유보 (신규 유보분 7,000원) |
| Vroong 계좌 잔액 | **3,000원** (< 신규 유보분 7,000원)   |
| 잔액에서 차감      | **0원** (부분 차감 없음)              |
| **미수금 이월**   | **7,000원**                     |

→ 잔액이 신규 유보분에 미치지 못하면 **이번에는 잔액 차감도 하지 않고 전액을 다음날로 이월**합니다.  
→ 3,000원을 부분 차감하는 방식은 사용하지 않습니다. (계산 일관성 유지)

---

## 전체 흐름 요약

```
매일 선정산 실행
    │
    ├─ ① 일차감 그리디 적용 (규칙별 전부-or-전무, 금액 큰 순)
    │       잔여 배달료 ≥ 규칙금액 → 해당 규칙 전액 차감   (Case 1)
    │       잔여 배달료 < 규칙금액 → 해당 규칙 전액 유보   (Case 2, 3)
    │
    ├─ ② 이월 미수금 있음? (일차감 후 잔여 배달료에서 회수)
    │       YES, 잔여 ≥ 미수금 → 전액 회수              (Case 4)
    │       YES, 잔여 < 미수금 → 가능한 만큼만 회수, 나머지 재이월  (Case 5)
    │       NO  → 생략
    │
    └─ 실지급 (항상 0원 이상)

    ③ 신규 유보분(오늘 차감 못한 일차감)이 있는 경우:
        계좌 잔액 ≥ 유보분 전액 → 잔액에서 전액 차감      (Case 6)
        계좌 잔액 < 유보분 전액 → 재이월 (다음날 미수금 누적) (Case 7)
```

---

## 핵심 원칙

1. **실지급액은 절대 음수가 되지 않습니다.**
2. **일차감은 배달료에서 먼저 처리되고, 이월 미수금 회수는 그 이후 잔여 배달료에서 진행됩니다.**
3. **일차감 규칙은 전부-or-전무: 배달료가 부족하면 해당 규칙 전액이 유보됩니다. (부분 적용 없음)**
4. **잔액 기반 차감은 오늘 신규 유보분에 대해서만 적용되며, 잔액이 유보분 전액 이상일 때만 실행됩니다.**
5. **미수금은 해소될 때까지 매일 이월됩니다.**
