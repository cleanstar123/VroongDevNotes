# 선정산 라이더 일차감 프로세스 안내

---

## 개요

선정산 라이더는 **매일 배달 완료 시점에 배달료를 지급**받습니다.  
이때 계약에 따라 설정된 **일차감(보험료, 장비 대여료 등)이 매일 배달료에서 차감**된 후 지급됩니다.

> 아래 모든 예시 기준: 일차감 단가 **7,000원 / 근무일**

---

## 기본 구조

```
오늘 배달료
- 이월 미수금 회수 (있는 경우 우선 차감)
- 오늘 일차감
= 실지급액
```

배달료가 충분하면 전액 차감 후 지급되지만,  
부족할 경우 가능한 금액만 차감하고 **나머지는 미수금으로 다음날 이월**합니다.

---

## 케이스별 예시

---

### Case 1. 정상 차감

> 오늘 배달료가 일차감보다 충분한 경우

| 항목 | 금액 |
|---|---|
| 오늘 배달료 | 80,000원 |
| 일차감 | 7,000원 |
| **실지급액** | **73,000원** |
| 미수금 발생 | 없음 |

→ 일차감 전액이 차감되고 73,000원이 지급됩니다.

---

### Case 2. 부분 차감 + 일부 유보

> 오늘 배달료가 일차감보다 적지만 0보다는 큰 경우

| 항목 | 금액 |
|---|---|
| 오늘 배달료 | 5,000원 |
| 일차감 | 7,000원 |
| 실제 적용된 차감 | 5,000원 (가능한 금액만) |
| **실지급액** | **0원** |
| **미수금 (내일로 이월)** | **2,000원** |

→ 배달료 전액(5,000원)까지만 차감하고, 부족한 **2,000원은 미수금으로 다음날 이월**됩니다.  
→ 실지급액은 0원 미만으로 내려가지 않습니다.

---

### Case 3. 전액 유보 (배달 없는 날)

> 오늘 배달이 없거나 배달료가 0인 경우

| 항목 | 금액 |
|---|---|
| 오늘 배달료 | 0원 |
| 일차감 | 7,000원 |
| 실제 적용된 차감 | 0원 |
| **실지급액** | **0원** |
| **미수금 (내일로 이월)** | **7,000원** |

→ 차감할 배달료가 없으므로 일차감 **전액이 미수금으로 다음날 이월**됩니다.

---

### Case 4. 이월 미수금 — 배달료에서 전액 회수

> 전날 미수금이 있고, 오늘 배달료가 충분한 경우

| 항목 | 금액 |
|---|---|
| 이월 미수금 | 2,000원 |
| 오늘 배달료 | 80,000원 |
| 이월 미수금 회수 | 2,000원 (배달료에서 우선 차감) |
| 오늘 일차감 | 7,000원 |
| **실지급액** | **71,000원** |
| 미수금 잔여 | 0원 (완전 회수) |

→ 배달료에서 이월 미수금(2,000원)을 먼저 회수한 뒤, 오늘 일차감(7,000원)을 추가로 차감합니다.

---

### Case 5. 이월 미수금 — 배달료가 부족하여 일부만 회수 후 재이월

> 이월 미수금이 있는데 오늘 배달료도 부족한 경우

| 항목          | 금액                                       |
| ----------- | ---------------------------------------- |
| 이월 미수금      | 7,000원                                   |
| 오늘 배달료      | 3,000원                                   |
| 이월 미수금 회수   | 3,000원 (배달료 전액으로 미수금 우선 회수)              |
| 오늘 일차감      | 7,000원 → 배달료 소진으로 전액 유보                  |
| **실지급액**    | **0원**                                   |
| **미수금 재이월** | **11,000원** (미회수 4,000원 + 오늘 일차감 7,000원) |

→ 배달료는 이월 미수금 회수에 우선 사용되고, 오늘 일차감은 배달료 부족으로 전액 유보됩니다.

---

### Case 6. 이월 미수금 — 잔액에서 전액 회수 (Full-or-Nothing)

> 배달료로 이월 미수금 회수가 불가할 때, Vroong 계좌 잔액을 활용하는 경우  
> **조건: 계좌 잔액 ≥ 이월 미수금 전액이어야만 차감 (부분 차감 없음)**

| 항목           | 금액                            |
| ------------ | ----------------------------- |
| 이월 미수금       | 7,000원                        |
| 오늘 배달료       | 0원 (배달 없음)                    |
| Vroong 계좌 잔액 | **10,000원** (≥ 이월 미수금 7,000원) |
| 잔액 차감        | 7,000원                        |
| **실지급액**     | **0원**                        |
| 미수금 잔여       | **0원** (완전 회수)                |

→ 배달료로 회수가 불가하더라도 **잔액이 미수금 전액 이상이면 잔액에서 전액 차감**하여 미수금을 해소합니다.

---

### Case 7. 이월 미수금 — 잔액도 부족하여 재이월

> 배달료도, 계좌 잔액도 이월 미수금을 커버하지 못하는 경우

| 항목           | 금액                           |
| ------------ | ---------------------------- |
| 이월 미수금       | 7,000원                       |
| 오늘 배달료       | 0원                           |
| Vroong 계좌 잔액 | **3,000원** (< 이월 미수금 7,000원) |
| 잔액 차감        | **0원** (부분 차감 없음)            |
| **미수금 재이월**  | **7,000원**                   |

→ 잔액이 부족할 경우 **이번에는 회수하지 않고 전액을 다음날로 재이월**합니다.  
→ 3,000원을 부분 차감하는 방식은 사용하지 않습니다. (계산 일관성 유지)

---

## 전체 흐름 요약

```
매일 선정산 실행
    │
    ├─ 이월 미수금 있음?
    │       YES → 오늘 배달료로 우선 회수 (가능한 만큼)
    │             여전히 미수금 남음 → 계좌 잔액 확인
    │                 잔액 ≥ 미수금 전액 → 잔액에서 전액 차감 (Case 6)
    │                 잔액 < 미수금 전액 → 다음날로 재이월   (Case 7)
    │
    ├─ 오늘 배달료(남은 금액) ≥ 일차감?
    │       YES → 전액 차감                     (Case 1, 4)
    │       NO  → 가능한 금액만 차감, 나머지 이월  (Case 2, 3, 5)
    │
    └─ 실지급 (항상 0원 이상)
```

---

## 핵심 원칙

1. **실지급액은 절대 음수가 되지 않습니다.**
2. **이월 미수금 회수는 오늘 배달료에서 일차감보다 우선 처리합니다.**
3. **잔액 기반 차감은 잔액이 미수금 전액 이상일 때만 실행합니다.** (부분 차감 없음)
4. **미수금은 해소될 때까지 매일 이월됩니다.**
